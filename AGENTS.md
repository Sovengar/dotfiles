# Dotfiles Repo — Agent Instructions

## Dependencies

When adding a new tool/dependency/package to any install script, **must also add it to `docs/ecosystem.md`** under the appropriate category.

## HyDE Context

Read `docs/hyde-architecture.md` before modifying any theme-related config (dunst, gtk, kitty, rofi, waybar). These files are partially autogenerated by HyDE's wallbash pipeline — only color variables change on theme switch, but the structure is fixed. Currently excluded from chezmoi via `.chezmoiignore`.
